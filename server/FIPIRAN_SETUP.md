# راه‌اندازی داده‌های فیپیران

پیش از انتشار نسخه جدید، دریافت اولیه یک‌سال را اجرا کنید:

```sh
cd /opt/rasad
python3 server/update_fipiran_data.py --days 366 --workers 4
```

پس از پایان دریافت اولیه، برنامه زمان‌بندی را نصب کنید:

```sh
crontab server/rasad-fipiran.cron
```

این برنامه در ساعت‌های ۰۰:۰۰، ۰۹:۰۰ و ۱۴:۰۰ به وقت ایران، هفت روز اخیر را دوباره دریافت می‌کند. مسیر `/opt/rasad` در فایل زمان‌بندی باید با مسیر واقعی پروژه روی سرور یکسان باشد.
